# Develops Code - Frontend Landing Page 🚀

Bem-vindo ao repositório oficial da **Develops Code**. Este projeto foi refatorado para uma arquitetura puramente Frontend, focada em performance, Clean Code e princípios SOLID.

---

## 🏗️ Visão Geral
A plataforma serve como o hub principal da Develops Code, apresentando serviços de desenvolvimento de software e capturando leads de potenciais clientes de forma elegante e interativa.

### Principais Funcionalidades:
- **Landing Page Premium**: Design responsivo com estética moderna e interações suaves.
- **Clean Architecture**: Código modularizado com separação clara de responsabilidades (SOLID).
- **Interações Dinâmicas**: Sistema de partículas, scroll reveal e FAQ interativo.
- **Formulário de Contato**: Interface refinada para captura de leads com tratamento de estados (envio/sucesso/erro).

---

## 🛠️ Stack Tecnológica

### Core
- **HTML5**: Estrutura semântica e acessível.
- **Vanilla CSS**: Design system customizado com variáveis CSS e micro-animações extraídas para arquivo dedicado.
- **Vanilla JavaScript**: Lógica modularizada (Module pattern) para gerenciamento de UI e interações, seguindo princípios SOLID.

### Bibliotecas Externas
- **Materialize CSS**: Base para componentes de UI e Grid.
- **Google Fonts**: Tipografia moderna (Syne & DM Sans).

---

## 🚀 Como Iniciar

O projeto é estático e pode ser aberto diretamente no navegador, mas recomendamos o uso de um servidor local para melhor experiência.

### 1. Clonar o Repositório
```bash
git clone https://github.com/lucabezerrapro/DevelopsCode.git
cd DevelopsCode
```

### 2. Executar Localmente
Você pode usar o script de desenvolvimento incluído (requer Node.js):
```bash
npm start
```
Ou simplesmente abrir o arquivo `index.html` em seu navegador.

---

## 📂 Estrutura do Projeto
- `index.html`: Estrutura principal da página, limpa de estilos e scripts inlined.
- `style.css`: Design system completo e estilização de componentes.
- `main.js`: Lógica de aplicação refatorada em módulos (FAQ, Navigation, Reveal, Particles, Form).
- `img/`: Ativos visuais e imagens.
